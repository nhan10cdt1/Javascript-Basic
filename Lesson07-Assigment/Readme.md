## Toán tử gán

---

| No. | Toán tử | ví dụ    | Tương đương  |
| --- | ------- | -------- | ------------ |
| 1   | =       | x = y    | x = y        |
| 2   | +=      | x +=y    | x = x + y    |
| 3   | -=      | x -=y    | x = x - y    |
| 4   | \*=     | x \*=y   | x = x \* y   |
| 5   | /=      | x /=y    | x = x / y    |
| 6   | \*\*=   | x \*\*=y | x = x \*\* y |
